學習王設定流程

Step 1 確認電力 - 請前往機房請確認您的電力狀態以及插座是否穩妥(已正常可跳過)
您可以確認當前區域的目前供電狀態以及停電公告台灣電力公司供電查詢
Step 2 NAS開機 - 按下主機開機鍵並等待10秒正常開機
確認disk燈號以及lan網孔的燈號皆呈現綠色閃爍

開機

  1. 按下開機鍵並等待10~20秒左右

確認燈號

  1. 聽到嗶聲
  2. STATUS, LAN1/LAN2, DISK1, DISK2 都有亮起綠色燈號，代表已正常開機
Step 3 開啟容器 - 查詢館內IP&容器開關
請先下載 " IP找尋助手 "IP找尋助手 ( Synology Assistant_檔案大小: 10.2MB ) 下載
或者是您也可以直接觀看 "教學影片"NAS系統開容器 教學影片

使用Assistant找尋您的IP

  1. 打開Synology Assistant
  2. 搜尋找到NAS主機名稱為「Study123」的NAS IP

開啟NAS管理系統中的容器

  1. 將您的NAS IP輸入到瀏覽器中
  2. 登入NAS控制頁面(帳號密碼若忘記請與資訊部索取)
  3. 進入docker>容器>開啟名為hls-ubuntu容器

特別注意：因為synology公司NAS系統更新的關係，有可能您的docker會改為Container Manager
Step 4 萬試通軟體 - 安裝/重裝軟體
" 若是場館有隨身碟，可以直接複製正常電腦的study123資料夾到異常的電腦，這樣是最快速方便的方式 "
請先下載 " Study123暫時性解決方案 壓縮包 "Study123暫時性解決方案 壓縮包 ( Study123暫時性解決方案_檔案大小: 809MB ) 下載免費好用的解壓縮工具 ( BandZip_檔案大小: 6.89MB ) 下載
或者是您也可以直接觀看 "教學影片"學習王安裝/重裝軟體 教學影片

解壓縮軟體包

  1. 將下載下來的Study123壓縮包雙擊點開
  2. 將Study123資料夾拖曳至桌面
  3. 等待解壓縮

修改config.txt的IP

  1. 進入 Study123 > Resorce > config.txt 雙擊點開
  2. 將您NAS的IP輸入影片中的範圍
  3. 儲存並離開config.txt

特別注意：您的NAS IP 為您在 IP查詢助手 查詢到的數字，並非教學中的192.168.0.9

在桌面建立萬試通捷徑

  1. 進入 Study123 > Release
  2. 找到 StudyCheckVersionII.exe 並「右鍵」點擊 選擇「傳送到」>「桌面(建立捷徑)」
  3. 即可使用萬試通
Step 5 補教系統 - 新增/檢視 學員帳號
請您先依照Step3確認您的NAS IP ( 若已經知道您的IP即可略過確認IP的步驟 ) Step3 確認您的 Study123 NAS IP

登入補教系統

  1. 將您的NAS IP輸入至瀏覽器中並在最後加上「:8002」，例如 192.168.0.9:8002
  2. 登入補教系統 (帳號密碼您若忘記請與資訊部人員索取)

新增 - 學員帳號

  1. 點選左上角「人員管理」 > 「學員人員管理」
  2. 滾輪滑至右面最底部，找到「新增學生帳號」並點擊
  3. 所有欄位皆須填入相應要求的內容
  4. 「行動電話」為您登入萬試通的密碼，請勿輸入超過10位數字 ( 建議您使用十個 1 )

特別注意：「生日」欄位預設值1994 5 15不代表已填入，您需要再次填入您自己設定的生日

檢視 - 學員帳號

  1. 點選左上角「人員管理」 > 「學員人員管理」
  2. 找到您需要登入的帳號，例如編號50號
  3. 點選該帳號欄位右側的「編輯資訊」進入編輯頁面
  4. 「行動電話」欄位則代表該帳號的密碼
避免資料庫中途因斷電導致損毀
請先下載 " IP找尋助手 "IP找尋助手 ( Synology Assistant_檔案大小: 10.2MB ) 下載

使用Assistant找尋您的IP

  1. 打開Synology Assistant
  2. 搜尋找到NAS主機名稱為「Study123」的NAS IP

關閉NAS管理系統中的容器

  1. 將您的NAS IP輸入到瀏覽器中
  2. 登入NAS控制頁面(帳號密碼若忘記請與資訊部索取)
  3. 進入docker>容器>關閉名為hls-ubuntu容器
  4. 確保名為hls-ubuntu容器開關呈現灰色

特別注意：因為synology公司NAS系統更新的關係，有可能您的docker會改為Container Manager

關機

  1. 請確認您已經關閉容器
  2. 長按電源鍵大約10~15秒左右直至燈號熄滅
  3. 確認所有燈號皆已熄滅
